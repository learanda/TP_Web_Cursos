Java para no programadores

Descripcion: En este curso se aprenderá desde cero a diseñar utilizando diferentes algoritmos de pseudocódigo y codificar pequeños sistemas. Partiendo desde los conceptos iniciales de programacion, como uso de variables y tipos de dato, aprenderás las estructuras básicas como sentencias condicionales, bucles de repetición, uso de vectores y una introducción a la interfaz gráfica de usuario para desarrollar aplicaciones de escritorio.Al finalizar este curso tendrás los conceptos fundamentales de Programacion para hacer el curso de Java Standard Programming.
